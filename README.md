# Cooperação que Transforma 🌱

> Uma landing page interativa bilíngue (PT/EN) que explica, de forma visual e acessível, como as cooperativas de crédito mantêm o dinheiro circulando nas comunidades locais — e por que isso importa.

**🔗 Demo ao vivo:** [cooperacao-que-transforma.vercel.app](https://cooperacao-que-transforma.vercel.app)

---

## 💡 Motivação

Cidades pequenas enfrentam um problema silencioso: o dinheiro entra, mas vai embora. Quando os moradores depositam em bancos tradicionais, o lucro gerado migra para sedes em grandes centros — São Paulo, Rio de Janeiro — beneficiando acionistas distantes.

As cooperativas de crédito invertem essa lógica: o excedente financeiro retorna para os próprios cooperados e para a comunidade. Este projeto nasceu do desejo de comunicar esse conceito de forma visual, simples e acessível para qualquer pessoa — com ou sem conhecimento financeiro.

---

## 🎯 Objetivo do projeto

- **Educacional:** Explicar a diferença entre bancos tradicionais e cooperativas de crédito de forma visual e não-técnica
- **Social:** Destacar o impacto do cooperativismo em pequenas e médias cidades brasileiras
- **Técnico:** Demonstrar domínio de HTML semântico, CSS moderno, JavaScript vanilla e deploy em produção

---

## ✨ Funcionalidades

- **Bilíngue PT/EN** — alternância instantânea com toggle de idioma
- **Animações SVG** — fluxo de dinheiro animado comparando banco tradicional vs. cooperativa
- **Counters animados** — dados reais do cooperativismo brasileiro com animação de contagem
- **Scroll reveal** — seções que aparecem suavemente conforme o usuário rola a página
- **Totalmente responsivo** — funciona em mobile, tablet e desktop
- **Zero dependências** — HTML, CSS e JavaScript puros, sem frameworks
